LeetCode 第九题 Palindrome Number

题目要求：

Determine whether an integer is a palindrome. Do this without extra space.

思路：

这个题目是来判定一个整数是不是回文？首先要明确什么是“回文”？

设N是一任意自然数，若将N的各位数字反向排列所得的自然数N1和N相等，则称N为一回文数。例如：N = 1234321

明确了”回文”的定义，这道题就变得很简单。

1 每次提取头尾两个数进行判断，看他们是否相等？

2 如果相等则去判断去掉头尾两个数的其他的；

3 如果头尾两个数不相等，那程序直接返回false
