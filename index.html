LeetCode 14 Longest Common Prefix

题目要求：Write a function to find the longest common prefix string amongst an array of strings

分析： 这个题是用来求一组字符串的最长公共前缀，[“asdffg”,”asdfrty”,”asdvbg”,”asdcv”,”asdffghh”]，这个数组中的最长公共前缀就是“asd”

解题思路：

  1. 首先确定所有字符串中最短的字符串，因为最长公共前缀一定是比最短的字符串短的

  2. 以第一个字符串为参考，与后面的字符串进行对比，比较其中的每一个字符是否相同

  3. 遇到不同的进行标记，记录下标，最后选择所有下标中最小的一个

设N是一任意自然数，若将N的各位数字反向排列所得的自然数N1和N相等，则称N为一回文数。例如：N = 1234321
